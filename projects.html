Tocky Talk

Tocky Talk is a native Android application developed using Android Studio. It contains learning modules on topics such as job interview, presentation and reading comprehension.

Features

  • Video lessons
  • Mindfulness exercises
  • Recorded practice interview activities
  • Multiple choice-type quizzes
  • Fill-in-the-blanks quizzes
  • Gamification system with badges, coins and items

Technologies Used

Android
Java
Android Studio
SQLite
Django

CLICK

CLICK (Computer Literacy Core Knowledge) is a standalone desktop application developed using LibGDX and Eclipse IDE. This app is designed to teach computer literacy to junior and senior high school students in the K-12 curriculum.

Features

  • Comics-type storytelling
  • Word processing activities
  • Multiple choice activities
  • Memory Game
  • Keyboard practice activities
  • Gamification system with hints and coins

Technologies Used

LibGDX
Java
SQLite
Apache POI

IMPACT

IMPACT (Introductory Modules on Problem Solving and Critical Thinking) is an educational desktop application developed using LibGDX and Eclipse IDE. It was designed as a visual novel game where the user is a newly hired fresh graduate who is just learning the ropes in his new job.

Features

  • Visual novel storytelling
  • Minigames
  • Tasks
  • Assessment
